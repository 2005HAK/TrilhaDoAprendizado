Destaques no texto

Podemos definir destaques do texto. Os destaques podem ser:
negrito,
texto destacado,
itálico,
texto com enfase.

A principio essas fomatações podem parecer iguais, mas o negrito e o italico, definidos respectivamente por: < b > e < i >, são somente elementos visuais, ou seja, vistos somente pelo usuário.

Já o texto destacado e o com enfase, definidos respectivamente por: < strong > e < em > serve como base para procura pelo navegador, como uma palavra-chave.

Para exibirmos tags devemos utilizar os comandos: < h1 >.

Alteração de fontes

Para alterar fontes devemos realizar esse procedimento via CSS, que serve especificamente para estilização de conteúdos.

Comando: < font color = "nome_da_cor" >

Exemplos:
Fontes alterada para azul
Fontes alterada para vermelho
Fontes alterada para verde
Fontes alterada para roxo
Fontes alterada para amarelo
Fontes alterada para cinza
