<table>

É a tag que permite contruir uma tabela definindo, dentro desta, as linhas e colunas.

As principais tags e atributos da tabela são:

<tr>
Representa uma linha da tabela, e contém as cédulas.
<td>
Representa uma cédula da tabela.
<th>
Definir as cédulas como cabeçalho. Podem ser definidas na primeira linha, ou na primeira coluna de cada linha para definir os títulos na vertical.
border
Não é uma tag, é um atributo utilizado na tag table para definir o tamanho da borda em volta da tabela.
nbsp
Está é a entidade que define um espaço em branco (ela possui um "&" antes do nbsp). Ela é utilizada porque o navegador pode não conseguir ler uma cédula vazia da maneira correta se simplesmente não houver nada dentro.
colspan
Faz a mesclagem de duas colunas. É utilizado na tag <th> e a sintaxe é: <th colspan="x">algo</th>, onde no lugar do "x" é inserido o número de colunas que serão mescladas.
rowspan
Segue o mesmo esquema do "colspan", com a diferença de que ao invés de mesclar colunas, irá mesclar linhas.

Estrutura:

<table border="1">
   <tr>
     <th>Cod.</th>
     <th>Nome</th>
     <th>telefone</th>
   </tr>

   <tr>
     <th>1</th>
     <th>José</th>
     <th>993485216</th>
   </tr>

   <tr>
     <th>2</th>
     <th>Espinócio</th>
     <th>&.nbsp</th>
   </tr>
</table>

Obs: O ponto entre o "&" e o "nbsp" é somente para que ele fique visível.

Exemplo da estrutura acima:

Cod. Nome Telefone
1 José 993485216
2 Espinócio  

<del>

Define que o texto foi excluido de um documento, mas é mantido para que se tenha um histórico da modificação que foi feita. Os navegadores normalmente riscam uma linha atravez do texto excluido.

Estrutura:

<p>Este texto vai usar a tag del <del>agora.</del></p>

Exemplo da estrutura acima:

Este texto vai usar a tag del agora.

<ins>

Define um texto que foi inserido em um documento. Ao utilizar em conjunto com a tag <del>, se tem noção do texto que foi retirado e qual foi inserido no lugar. Os navegadores normalmente sublinham o texto inserido.

Estrutura:

<p>Este texto vai usar a tag <del>del</del> <ins>ins</ins> agora.</p>

Exemplo da estrutura acima:

Este texto vai usar a tag del ins agora.

<sub>

Define um texto como subscrito, aparecendo meio caracter abaixo da linha normal e com letras menores. É muito utilizado para representar fórmulas químicas.

Estrutura:

<p>Fórmula da água: H<sub>2</sub>O</p>

Exemplo da estrutura acima:

Fórmula da água: H2O

<sup>

Define um texto como sobrescrito, aparecendo meio caracter acima da linha normal e em caracter menor. Utilizado, por exemplo, para escrever formulas matemáticas que possuem potenciação, ou indicar um significado (como nos livros).

Estrutura:

<p>Tim Berners-Lee  inventou o WWW<sup>[3]</sup></p>
<p><sup>[3]</sup>World Wide Web (em português, Rede de Alcance Mundial).</p>

Exemplo da estrutura acima:

Tim Berners-Lee inventou o WWW[3]

[3]World Wide Web (em português, Rede de Alcance Mundial).

<blockquote>

Especifica uma seção que é citada de outra fonte, aparecendo recuado em relação ao texto normal. Usado em citações longas.

Estrutura:

<blockquote cite="https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/blockquote">

   O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é 
   uma longa citação. Normalmente, este é processado visualmente pelo recuo (ver Notas sobre como mudá-lo).
   A URL para a fonte da citação pode ser dada usando o atributo cite, enquanto uma representação de
   texto da fonte pode ser dada usando o <cite> elemento.

</blockquote>

Exemplo da esrtutura acima:

O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo (ver Notas sobre como mudá-lo). A URL para a fonte da citação pode ser dada usando o atributo cite, enquanto uma representação de texto da fonte pode ser dada usando o <cite> elemento.

<q>

Utilizado para pequenas citações. Não recua o texto, somente insere aspas nas duas pontas.

Estrutura:

<p>O site W3Schools diz que <q cite="https://www.w3schools.com/tags/tag_q.asp">A <q>tag define uma cotação curta</q>.</p>

Exemplo da estrutura acima:

O site W3Schools diz que A <q>tag define uma citação curta.

Obs: O atibuto "cite" é utilizado tanto na tag "blockquote" quanto na "q", e seu valor designa uma URL fonte da informação, ou uma mensagem para a informação citada.

<abbr>

Define uma abreviação de uma palavra, com "HTML" por exemplo. Utilizando o atributo title na tag <abbr>, ao passar o mouse por cima, a descrição definida neste atributo aparecerá.

Estrutura:

<p>Uso da tag <abbr title="Tag de abreviação"><abbr></abbr></p>

Exemplo da estrutura acima:

Uso da tag <abbr>

<address>

Usado para inserir informações de contato do autor de um documento ou artigo, como: e-mail, URL, telefone, etc. Os navegadores normalmente deixam o texto que esta dentro da tag <addrress> em itálico, e uma quebra de linha é adicionada antes e depois da tag.

Estrutura:

<address>
   Contato: <a href="mailto:hebertakubis@outlook.com">Hebert A. Kubis</a>.<br>
   Site:<br>
   www.hak.com<br>
   Irati, PR<br>
   Brasil
</address>

Exemplo da estrutura acima:

Contato: Hebert A. Kubis.
Site:
www.hak.com
Irati, PR
Brasil

<cite>

Utilizado para definir um nome de uma arte, por exemplo: um poema, uma musica, um filme, etc. Não é utilizado no nome do autor, somente no título da obra.

Estrutura:

<figure>

   <img src="img/Redes de computadores" width="200px" alt="Redes de computadores">

</figure>

<p><cite>Redes de computadores</cite> de Andrew Tanenbaum, Nick Feamster e David Wetherall. 1ª publicação em 1988.</p>

Exemplo da estrutura acima:

Redes de computadores de Andrew Tanenbaum, Nick Feamster e David Wetherall. 1ª publicação em 1988.

<bdo>

Utilizada para mudar a direção do texto.

Estrutura:

<p>O texto abaixo irá da direita para a esquerda<br>
<bdo dir="rtl">Olha só como é verdade</bdo><br>
<bdo dir="ltr">E esse é normal.</bdo></p>

Exemplo da estrutura acima:

O texto abaixo irá da direita para a esquerda:
Olha só como é verdade
E esse é normal

<code>

Define que um trecho de um texto é um código de computador.

Estrutura:

<p>Em CSS o <code>background-color</code> define o fundo de um elemento, como por exemplo o findo desta pagina.</p>

Exemplo da estrutura acima:

Em CSS o background-color define o fundo de um elemento, como por exemplo o fundo desta página.

<pre>

Texto pré-formatado, ou seja, tudo oque for escuito dentro de uma tag <pre> será exibido do mesmo modo que foi escrito, preservando inclusive as quebras de linha.

Estrutura:

<p><pre>
Este
é
um texto
escrito em
uma tag
pre
</pre></p>

Exemplo da estrutura acima:

            Este
            é
            um texto
            escrito em
            uma tag
            pre

Obs: Também persebe-se que o texto adiquire o mesmo recuo que possui dentro do documento HTML

<kbd>

Utilizado para definir uma entrada de teclado.

Estrutura:

<p>Para acessar o executar do Windows precione as teclas <kbd>Win</kbd> + <kbd>R</kbd>.</p>

Exemplo da estrutura acima:

Para acessar o executar do Windows precione as teclas Win + R.
