Hyperlinks

São elementos dentro de um Hipertexto que permitem pular de um ponto a outro do mesmo documento, ou redirecionar o usuário a outro documento.
A tag que define um hiperlink é: <a>

Sintaxe: <a href="url" target="_self">texto que conterá o hiperlink</a>

OBS: O texto que conterá o hiperlink para o redirecionamento não precisa necessáriamente ser um texto, pode ser qualquer outro elemento do documento HTML, como por exemplo uma imagem.

Elementos:

  • href

    • É o atributo onde fica o link para o qual o usuário será redirecionado ao clicar.

  • texto que conterá o link

    • É o elemento que fica visível ao usuário e, ao passar o mouse por cima, a seta da a opção de clicar no elemento.

  • target

    • É o atributo que especifica onde o link sera aberto, possuindo alguns valores, como:

    • _self

      • Modo padrão, mesmo que o atributo target não seja especificado, será esse o valor utilizado. Abrirá o ducumento na mesma janela em que foi clicado.

    • _blank

      • Abre o link em uma nova janela.

Exemplos:

  1. Link com abertura em uma nova guia: Youtube

    2. Sintaxe: <a href="https://www.youtube.com/channel/UC9JwOUxGWyfkbatL92SCevw" target="_blank">Youtube</a>

  2. Link com abertura na mesma guia: Mesma Guia

    3. Sintaxe: <a href="ExemploMesmaGuia.html">Mesma Guia</a>

    Nesse exemplo, de abertura na mesma guia, também foi utilizado a possibilidade de abrir um outro documento HTML, utilizando no lugar do URL o endereço da pasta: "ExemploMesmaGuia.html". Ou se não tivesse na mesma hierarquia: nome_da_pasta/ExemploMesmaGuia.html.

  3. Link para fazer o redirecionamento no mesmo documento: Inicio

    4. Nesse caso o link envia o usuário para uma parte no mesmo documento, e para isso o lugar onde será redirecionado precisa estar marcada, como por exemplo o titulo hiperlink no começo da página, ele foi marcado da seguinte forma: <h1 id="inicio">Hyperlinks</h1>. Foi colocado um id na tag de título, e poderia ter sido colocado em qualquer outra tag, como uma tag <p> por exemplo.

    Também pode ser utilizado a seguinte sintaxe: <h1><a name="inicio"></a>Hyperlinks</h1>, mas deste modo é utilizado uma tag a mais, portanto o mais pratico é utilizar id.

    E para referenciar esse id que está em uma das tags do documento, uma hashtag(#) deve ser colocada antes do nome, desta forma: <a href="#inicio">Inicio</a>

  4. Link com imagem:

    5. Pode redirecionar para uma outra página.

    Pode abrir a própria imagem em uma nova guia e fazer qualquer outro redirecionamento, seguindo a mesma sintaxe dos outros exemplos, com a diferença de que no lugar do texto que conteria o link é colocado uma imagem.
